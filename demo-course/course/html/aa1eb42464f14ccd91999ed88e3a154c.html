Note: Configuration Required

The Scorm X-block is not included by default in this course because it is not installed on some older versions of the Open edX® LMS, and therefore it has the potential to cause an error when importing the course. There are two pieces of configuration that may be required to view the SCORM demonstration. If you choose not to do either, then you may also review the video to see the functionality. 

SCORM XBlock

The SCORM X-block may not be installed by default on your instance of the Open edX® platform, especially if you have an instance older than Nutmeg. If it is not installed, then you will not have the option to add a SCORM component in your Advanced components list in Studio. If you would like to install the SCORM XBlock, follow the documentation.

SCORM Demonstration Package

If the SCORM X-block is installed, then you will be able to add the SCORM x-block. You will next need to upload the SCORM package in studio. The package can be downloaded here. Review the video to see how to upload the package into the SCORM XBlock.
